1) Compiling the Java file: .\javac program.java
.\javac EM.java

2) Running the Java Code: .\java program <inputDataFile> <K>
.\java EM em_data.txt 3


**Two Heuristics are implemented in the code.By setting the h=1 or h=2 in the code,
we can switch between the heuristic.
